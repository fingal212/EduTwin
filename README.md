# 🎓 EduTwin — Ваш персональный цифровой двойник в обучении

[![Netlify Status](https://api.netlify.com/api/v1/badges/default/deploy-status)](https://s4uleserikovna.netlify.app)

**EduTwin** — это инновационная образовательная платформа, созданная в рамках хакатона «BilimHack Almaty». Проект решает кейс **«AI-агент для тестирования и наставничества»**, автоматизируя экзамены и превращая тестирование в мощный инструмент обучения.

🔗 **[Посмотреть рабочий проект (Live Demo)](https://s4uleserikovna.netlify.app)**

## 💡 О проекте
Мы разработали систему, которая объединяет генерацию тестов, автоматическую проверку и интеллектуальное наставничество в единый бесшовный процесс. 

Вместо стандартных статических тестов, EduTwin предлагает:
1. **Генерацию на лету:** ИИ создает уникальные тесты по выбранному предмету и уровню сложности.
2. **Мгновенную проверку:** Автоматический подсчет баллов и выявление слабых зон студента.
3. **Интеллектуальный разбор ошибок:** AI-ассистент (Saule AI) анализирует неправильные ответы и подробно объясняет концепции, которые нужно повторить.

## 🚀 Ключевые функции
* 🧠 **Saule AI Assistant** — умный чат-бот на базе Gemini 2.0 Flash, способный отвечать на вопросы, проверять код/тексты и даже распознавать загруженные изображения (Vision).
* 📝 **Умное тестирование** — динамическая генерация JSON-тестов через API нейросети.
* 👩‍🏫 **Инструменты преподавателя** — автоматическое создание планов уроков и панель аналитики успеваемости.
* 🎥 **База знаний** — встроенный видеоплеер с фильтрацией лекций по категориям.
* 🔄 **SkillChain** — интегрированный модуль для P2P-обмена навыками между студентами.

## 🛠 Технологический стек
* **Frontend:** HTML5, CSS3, Vanilla JavaScript, FontAwesome.
* **Backend / API:** Netlify Serverless Functions (Node.js).
* **AI Engine:** Google Gemini 2.0 Flash API (Text & Vision capabilities).
* **Деплой:** Netlify.
